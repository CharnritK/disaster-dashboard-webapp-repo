# Layout, Titles, And Metadata

Dashboard cards should be scannable and review-friendly.

## Chart Frame

Every chart card should expose:

- title as a heading;
- subtitle or context line;
- quality status text;
- source note or caveat when needed;
- screen-reader summary;
- chart body.

## Titles

Prefer titles that name the metric and comparison:

- Good: `Response Gap Percent by District Name`
- Weak: `District chart`

## Subtitles

Use subtitles for unit, geography, time scope, or mobile reduction. Do not use subtitles as generic decoration.

## Caveats

If a chart uses a risky field, keep the caveat in the visual card and downstream export context.
