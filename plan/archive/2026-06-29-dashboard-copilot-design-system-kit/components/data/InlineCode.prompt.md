Data terms are typographically marked: wrap every column or dataset name in `InlineCode`, usually with a human display label alongside. Never use it for emphasis — only for literal data identifiers. 5px radius (smaller than the 8px standard), monospace, scales with surrounding text via `0.88em`.

```jsx
<p style={{ color: "var(--muted)", margin: 0 }}>
  Need severity score (<InlineCode>need_severity_score</InlineCode>) is 18% missing in East District.
</p>
```
