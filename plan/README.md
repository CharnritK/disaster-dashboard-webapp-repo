# Plan And Handoff Index

This folder contains planning, handoff, and archived reference material for
Dashboard Copilot. Public readers should treat `README.md`, `docs/`, `db/`,
`specs/`, and the active package below as the current source of truth.

## Current

- `dashboard_copilot_codex_handoff_v1_1/`: active controlled-beta execution
  package, task history, QA evidence, release gates, staging runbooks, and
  remaining approval boundaries.

## Foundation Reference

- `final_handoff_package/`: earlier foundation and release-gate context retained
  for traceability. It is not the active execution package.

## Historical Archive

- `archive/`: old Codex prompt packs, design kits, and handoff bundles retained
  for traceability only.

Archived folders are not the active product plan. If an archived instruction
conflicts with `AGENTS.md`, `README.md`, `docs/`, `specs/`, or
`dashboard_copilot_codex_handoff_v1_1/`, use the current files.
