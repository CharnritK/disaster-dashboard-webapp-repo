# Stop Conditions

Stop and ask for human review if:

- Wrong branch.
- Unexpected dirty working tree.
- Tests cannot be defined for a feature.
- A goal requires new dependencies.
- A change could send full uploaded rows externally.
- A sample or fixture contains real PII.
- A secret appears.
- Build fails after two focused fix attempts.
- Red-team blocks a test or finds it can be trivially bypassed.
- The demo path breaks after optional work.
- Scope expands into DuckDB/dlt/Rill/LangGraph/new app.
