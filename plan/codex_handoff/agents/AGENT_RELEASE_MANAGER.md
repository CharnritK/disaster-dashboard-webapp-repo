```bash
npm run lint
npm run test
npm run build
```

## Manual smoke path
- Open app.
- Use Response prioritization template.
- Download suggested CSV template.
- Load fragmented demo data.
- Profile data.
- Review Evidence Coverage Map.
- Harmonize data.
- Accept or adjust join.
- Review decision readiness.
- Generate dashboard.
- Export CSV/PDF/PNG/handoff log.
- Load risky sample.
- Confirm `decision_unsafe` and soft-gated dashboard generation.
