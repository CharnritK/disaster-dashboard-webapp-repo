## Narration Transcript

In the public demo, the user starts with the decision, not the visualization.
They choose the response-prioritization template, then load bundled fragmented
sample data instead of uploading sensitive files. The app profiles fields,
missingness, and evidence coverage. It recommends a join, but the user reviews
the key and cleaning actions before accepting. Readiness becomes a checkpoint,
not approval. The dashboard keeps source notes and caveats visible, and the
final handoff package preserves context for the response team.
